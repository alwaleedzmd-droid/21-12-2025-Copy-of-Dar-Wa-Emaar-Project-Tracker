# تعليمات تثبيت مكتبات التصدير

## المكتبات المطلوبة
تم إضافة خاصية تصدير لوحة الإحصائيات إلى ملفات Excel و PDF.

## خطوات التثبيت

### الطريقة الأولى: تثبيت جميع المكتبات
```bash
npm install
```

### الطريقة الثانية: تثبيت المكتبات الجديدة فقط
```bash
npm install jspdf html2canvas
```

## المكتبات المضافة
- **jspdf**: لتصدير الصفحة إلى ملف PDF
- **html2canvas**: لتحويل HTML إلى صورة (مطلوب لـ PDF)
- **xlsx**: موجود مسبقاً في المشروع

## الاستخدام
بعد التثبيت، ستظهر زرين جديدين في أعلى لوحة الإحصائيات:
1. **تصدير Excel**: لتحميل البيانات في ملف Excel منظم بأربعة أوراق
2. **تصدير PDF**: لتحميل صورة كاملة من لوحة الإحصائيات

## المشاكل المحتملة
إذا واجهت مشكلة في تصدير PDF، تأكد من:
- تثبيت المكتبات بشكل صحيح
- عدم وجود أخطاء في Console المتصفح
- استخدام متصفح حديث (Chrome, Firefox, Edge)

## محتوى ملف Excel المصدّر
1. **ملخص الإحصائيات**: الأرقام الرئيسية
2. **الطلبات الفنية**: تفاصيل حسب الحالة
3. **طلبات الإفراغات**: تفاصيل حسب الحالة
4. **توزيع الأعمال**: تفاصيل لكل مشروع
